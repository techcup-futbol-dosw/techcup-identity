# Microservicio de Identidad

## Descripción

El Microservicio de Identidad es responsable de gestionar la autenticación, autorización y control de acceso de los usuarios dentro del sistema.

Este servicio permite registrar usuarios, gestionar roles y permisos, autenticar usuarios mediante credenciales y generar tokens JWT que permiten el acceso seguro a los demás microservicios del sistema.

Además, este microservicio implementa mecanismos de auditoría para registrar eventos importantes relacionados con la seguridad.


## Responsabilidades del servicio

El microservicio de identidad tiene las siguientes responsabilidades principales:

- Registro de usuarios
- Autenticación de usuarios
- Generación de tokens JWT
- Validación de tokens JWT
- Gestión de roles
- Gestión de permisos
- Control de acceso basado en roles
- Gestión de sesiones
- Registro de auditoría de eventos
